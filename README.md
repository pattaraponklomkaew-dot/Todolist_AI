# Todo List Application

โปรแกรมจัดการรายการสิ่งที่ต้องทำ (Todo List) แบบ Command Line Interface (CLI) ที่ช่วยให้คุณสามารถจัดการงานต่างๆ ได้อย่างมีประสิทธิภาพ

A command-line Todo List application for managing your tasks efficiently.

## คุณสมบัติ (Features)

- เพิ่มงานใหม่ (Add new tasks)
- แสดงรายการงานทั้งหมด (View all tasks)
- แก้ไขรายละเอียดงาน (Edit task details)
- ลบงาน (Delete tasks)
- บันทึกข้อมูลอัตโนมัติ (Auto-save feature)
- รองรับภาษาไทย (Thai language support)

## การติดตั้ง (Installation)

1. ต้องมี Python 3.6 หรือใหม่กว่า
2. Clone repository นี้:
```bash
git clone https://github.com/[your-username]/TodoList_AI.git
cd TodoList_AI
```

## การใช้งาน (Usage)

รันโปรแกรมด้วยคำสั่ง:
```bash
python main.py
```

### เมนูหลัก (Main Menu)
```
=== Todo List Menu ===
1. เพิ่มงานใหม่
2. ดูงานทั้งหมด
3. แก้ไขงาน
4. ลบงาน
5. ออกจากโปรแกรม
```

### ตัวอย่างการใช้งาน (Examples)

1. **เพิ่มงานใหม่:**
   ```
   เลือกเมนู: 1
   ชื่อเรื่อง: ทำการบ้านคณิตศาสตร์
   รายละเอียด: ทำแบบฝึกหัดบทที่ 5
   วันครบกำหนด: 2025-12-31
   ```

2. **ดูรายการงาน:**
   ```
   เลือกเมนู: 2
   
   === รายการงานทั้งหมด ===
   ลำดับ | ชื่องาน | วันครบกำหนด | สถานะ
   --------------------------------------------------
   1. ทำการบ้านคณิตศาสตร์ | 2025-12-31 | ✗ ยังไม่เสร็จ
   ```

3. **แก้ไขงาน:**
   ```
   เลือกเมนู: 3
   กรุณาเลือกลำดับงานที่ต้องการแก้ไข: 1
   
   === เมนูแก้ไขงาน ===
   1. แก้ไขชื่อเรื่อง
   2. แก้ไขรายละเอียด
   3. เปลี่ยนสถานะงาน
   ```

4. **ลบงาน:**
   ```
   เลือกเมนู: 4
   กรุณาเลือกลำดับงานที่ต้องการลบ: 1
   ต้องการลบงานนี้จริงหรือไม่ (y/n): y
   ```

## การจัดเก็บข้อมูล (Data Storage)

- ข้อมูลทั้งหมดจะถูกบันทึกในไฟล์ `tasks.json`
- โปรแกรมจะโหลดข้อมูลอัตโนมัติเมื่อเริ่มต้น
- บันทึกข้อมูลอัตโนมัติเมื่อออกจากโปรแกรม

## โครงสร้างข้อมูล (Data Structure)

แต่ละงานจะประกอบด้วยข้อมูลต่อไปนี้:
```json
{
  "id": 1,
  "title": "ชื่องาน",
  "description": "รายละเอียด",
  "due_date": "YYYY-MM-DD",
  "completed": false
}
```

## ข้อกำหนดของระบบ (System Requirements)

- Python 3.6+
- ระบบปฏิบัติการ: Windows, macOS, หรือ Linux
- รองรับการแสดงผลภาษาไทย (UTF-8 encoding support)
